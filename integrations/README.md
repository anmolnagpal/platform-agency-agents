# Integrations

Run `./scripts/convert.sh` to regenerate these directories, then `./scripts/install.sh` to copy them into supported tools.

- `claude-code/` – individual `.md` agents
- `antigravity/` – skill folders with `SKILL.md`
- `gemini-cli/` – extension-style skills
- `opencode/` – `.md` agents ready for `.opencode/`
- `cursor/` – `.mdc` rules per agent
- `aider/` – merged `CONVENTIONS.md`
- `windsurf/` – `.windsurfrules`

Directories will be overwritten each time you run `convert.sh`.
